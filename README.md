# Animatch 🐾

App interactiva para que los más pequeños aprendan los animales. Navega entre tarjetas, escucha el nombre del animal y reproduce su sonido real.

## ✨ Funcionalidades

- **Tarjetas con imagen real** de cada animal
- **Navegación con botones** ◀ ▶ para pasar de animal
- **Lectura del nombre** con voz amable (Laura, en español)
- **Sonido real del animal** 🎶
- **Fondos temáticos** que cambian según el grupo del animal
- **Transiciones suaves** pensadas para niños pequeños

## 🐾 Grupos de animales

| Grupo | Animales |
|-------|----------|
| 🐾 Domésticos | Perro, Gato, Pez, Conejo, Tortuga, Loro |
| 🐮 Granja | Vaca, Oveja, Caballo, Cerdo, Gallina, Pato |
| 🦁 Selva | León, Elefante, Mono, Jirafa, Tigre, Cebra |
| 🐬 Marinos | Delfín, Ballena, Pez Payaso, Tortuga Marina, Foca |

## 🛠️ Stack

- **Vanilla JS** con Web Components
- **Vite** como bundler
- **CSS** con Flexbox, transiciones, y Shadow DOM
- **Web Speech API** para la lectura del nombre
- **GitHub Pages** para el despliegue

## 📦 Desarrollo local

```bash
pnpm install
pnpm run dev      # servidor de desarrollo
pnpm run build    # build de producción
pnpm run deploy   # despliegue a GitHub Pages
```
