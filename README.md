# Animatch 🐾

App interactiva para que los más pequeños aprendan los animales. Desliza tarjetas al estilo Reigns, escucha el nombre del animal y su sonido.

## Grupos

- 🐾 Domésticos
- 🐮 Granja
- 🦁 Selva
- 🐬 Marinos

## Stack

- Vanilla JS + Web Components
- Vite
- CSS custom properties

## Desarrollo

```bash
pnpm install
pnpm run dev
```

## Deploy

```bash
pnpm run build
pnpm run deploy
```
